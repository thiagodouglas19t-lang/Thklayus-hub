# THKLAYUS Hub

App de serviços digitais + pedidos + suporte + painel ADM.

A proposta principal não é mais ser apenas uma plataforma de cursos. O app agora funciona como um hub para vender e acompanhar pedidos simples que combinam com o que o criador realmente consegue entregar: artes geradas com IA, ideias visuais, capas, posts, logos simples, prompts, packs digitais e suporte básico.

Importante: o app não deve prometer apresentações, slides completos ou edição avançada de imagens enquanto isso não for domínio real do projeto.

Cursos e materiais continuam existindo como extras/bônus, mas não são mais o centro do produto.

Stack: React + Vite + TypeScript + Tailwind + Supabase + Vercel.
