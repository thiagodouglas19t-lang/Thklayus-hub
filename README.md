# THKLAYUS Hub

App de serviços digitais + pedidos + suporte + painel ADM.

A proposta principal não é mais ser apenas uma plataforma de cursos. O app agora funciona como um hub para vender e acompanhar serviços simples, como apresentações escolares, artes, materiais digitais, pedidos personalizados e suporte.

Cursos e materiais continuam existindo como extras/bônus, mas não são mais o centro do produto.

Stack: React + Vite + TypeScript + Tailwind + Supabase + Vercel.
