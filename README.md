# THKLAYUS Hub

App de serviços digitais + pedidos + suporte + painel ADM.

A proposta principal não é mais ser apenas uma plataforma de cursos. O app agora funciona como um hub para vender e acompanhar pedidos simples que combinam com o que o criador realmente consegue entregar:

- apresentações simples feitas com ajuda de IA;
- slides básicos para escola, família ou pequenos projetos;
- artes geradas com IA;
- capas, posts, imagens de perfil e wallpapers;
- logos simples/ideias visuais;
- prompts, packs digitais e suporte básico.

Transparência importante: apresentações podem usar imagens ilustrativas ou geradas por IA. Quando fizer sentido, as imagens podem ser trocadas manualmente por fotos reais de bancos gratuitos. O app não deve prometer edição avançada de imagem nem imagem real perfeita em todos os casos.

Cursos e materiais continuam existindo como extras/bônus, mas não são mais o centro do produto.

Stack: React + Vite + TypeScript + Tailwind + Supabase + Vercel.
